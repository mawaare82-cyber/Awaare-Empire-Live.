# Awaare Empire Live

Kani waa App-ka menu-ka casriga ah ee **Awaare Empire**. Waxaa loogu talagalay in macaamiishu ay si degdeg ah oo qurux badan ugu arkaan noocyada cabitaannada iyo juuska aan bixinno.

## Astaamaha
* **Modern Interface:** Naqshad casri ah oo ku habboon mobile-ka.
* **Categories:** Waxaan u qaybinnay menu-ka:
    * **Modern Drinks** (Mint Mojito, Smoothies, etc.)
    * **Traditional Juices** (Mango Papaya, Lemon Mint, etc.)

* [Tailwind CSS](https://tailwindcss.com/)
